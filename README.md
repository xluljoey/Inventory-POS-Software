# Inventory Management System

This is an inventory management system built with Flet for Python.

## Setup and Installation

1. The virtual environment is already created in the `inventory_env` directory
2. Install or update dependencies:
   ```bash
   inventory_env/bin/pip install -r requirements.txt
   ```

3. Run the application:
   ```bash
   python run_app.py
   ```
   
   Or alternatively:
   ```bash
   inventory_env/bin/python main.py
   ```

## Fedora 43 Compatibility

If you're running on Fedora 43 and experiencing blank windows, please see the detailed troubleshooting guide in README_FEDORA_ISSUE.md.

## Running with Compatibility Workarounds

For Fedora 43 users, use the special run script which includes compatibility workarounds:
```bash
python run_app.py
```

This will run the application in the virtual environment with the necessary environment variables set to address the blank window issue.

## Virtual Environment

The project uses a virtual environment located at `inventory_env/` to manage dependencies. Always use this environment when running the application to ensure compatibility and proper dependency management.
